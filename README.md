# AI Customer Support Automation System

## Overview

An AI-powered customer support system built using FastAPI, Streamlit, SQLite, Machine Learning, NLP, and RAG.

## Features

- Ticket Classification
- Sentiment Analysis
- Escalation Logic
- RAG Retrieval
- Automated Response Generation
- Ticket History
- Analytics Dashboard
- CSV Export
- Search & Filters

## Tech Stack

- Python
- FastAPI
- Streamlit
- SQLite
- Scikit-learn
- TextBlob
- Pandas
- Matplotlib

## Run Backend

```bash
uvicorn backend.main:app --reload
